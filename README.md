# Usage

```
Usage: filelist.py <options> <object>

Options:
  -v, --version         查看版本号
  -h, --help            查看帮助信息
  -f FILENAME, --file=FILENAME
                        指定查找文件 或者关键词
  -d DIRECTORY, --directory=DIRECTORY
                        指定查找目录如果没有指定那就是当前目录
  -n NUM, --file_num=NUM
                        指定只要查找几个文件
```
